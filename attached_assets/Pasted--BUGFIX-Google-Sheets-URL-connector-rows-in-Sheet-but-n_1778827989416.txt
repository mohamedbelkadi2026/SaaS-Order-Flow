🐛 BUGFIX — Google Sheets URL connector: rows in Sheet but no orders in TajerGrow
Symptom
User connected a public Google Sheet URL successfully. Added a test row to the sheet. Waited 5+ minutes. Sheet shows the row, but TajerGrow Mes Commandes → Nouveaux shows nothing. No order created.
Multiple possible causes; this prompt addresses all of them defensively.
Suspected root causes (in order of likelihood)

Cron not registered — setInterval for syncAllPublicSheets missing or broken in server/index.ts
Header detection too aggressive — sample data row mis-detected as header, gets skipped
lastSyncState initialized wrong — first sync stores rows.length BEFORE creating orders, so all rows look "already synced"
magasinId missing on order — order created but hidden by Mes Commandes magasin filter
Column mapping not used — cron ignores user-defined mapping, reads fixed positional columns
Cron throws silently — exception swallowed without logging

Fixes (apply all)
Fix 1 — Ensure cron is registered AND logs every tick
In server/index.ts, near the bottom (after server starts), ensure this code exists:
tsimport { syncAllPublicSheets } from './cron/sync-gsheets-public';

const GSHEETS_CRON_INTERVAL_MS = 5 * 60 * 1000;   // 5 minutes

// Initial sync after 30 seconds (gives DB time to settle)
setTimeout(() => {
  console.log('[GSHEETS-PUBLIC-CRON] Initial sync starting...');
  syncAllPublicSheets()
    .then(() => console.log('[GSHEETS-PUBLIC-CRON] Initial sync complete'))
    .catch(err => console.error('[GSHEETS-PUBLIC-CRON] Initial sync failed:', err.message, err.stack));
}, 30_000);

// Recurring sync every 5 minutes
setInterval(() => {
  const startTime = Date.now();
  console.log(`[GSHEETS-PUBLIC-CRON] Tick starting at ${new Date().toISOString()}`);
  syncAllPublicSheets()
    .then(() => {
      const elapsed = Date.now() - startTime;
      console.log(`[GSHEETS-PUBLIC-CRON] Tick complete in ${elapsed}ms`);
    })
    .catch(err => console.error('[GSHEETS-PUBLIC-CRON] Tick failed:', err.message, err.stack));
}, GSHEETS_CRON_INTERVAL_MS);

console.log(`[GSHEETS-PUBLIC-CRON] Registered — initial sync in 30s, then every ${GSHEETS_CRON_INTERVAL_MS / 1000}s`);
The console.log lines are critical — without them, you can't tell from Railway logs whether the cron is running.
Fix 2 — syncAllPublicSheets must log every connection it processes
In server/cron/sync-gsheets-public.ts, replace the top-level syncAllPublicSheets function with this version that logs liberally:
tsexport async function syncAllPublicSheets() {
  const conns = await db.select().from(storeIntegrations)
    .where(and(
      eq(storeIntegrations.provider, 'gsheets'),
      eq(storeIntegrations.status, 'active'),
      sql`${(storeIntegrations as any).gsheetUrl} IS NOT NULL`,
    ));

  console.log(`[GSHEETS-PUBLIC] Found ${conns.length} active connection(s)`);
  
  if (conns.length === 0) {
    console.log('[GSHEETS-PUBLIC] No active gsheets connections — skipping');
    return;
  }

  for (const conn of conns) {
    try {
      console.log(`[GSHEETS-PUBLIC] Processing connection id=${conn.id} store=${conn.storeId} magasin=${(conn as any).magasinId || 'NULL'} sheet=${(conn as any).gsheetId}`);
      await syncOnePublicSheet(conn as any);
    } catch (err: any) {
      console.error(`[GSHEETS-PUBLIC] Connection ${conn.id} FAILED:`, err.message, err.stack);
    }
  }
}
Fix 3 — Verbose logging inside syncOnePublicSheet
In the same file, add detailed logging to syncOnePublicSheet:
tsasync function syncOnePublicSheet(conn: any) {
  const sheetId = conn.gsheetId;
  const tabs = conn.gsheetTabs || [];
  const state = conn.gsheetSyncState || {};
  const mapping = conn.gsheetColumnMapping || {
    name: 0, phone: 1, city: 2, address: 3,
    product: 4, price: 5, quantity: 6,
    note: 7, utmCampaign: 10, utmSource: 11, productId: 12,
  };

  console.log(`[GSHEETS-PUBLIC] sheet=${sheetId} tabs=${tabs.length} mapping=${JSON.stringify(mapping)}`);
  console.log(`[GSHEETS-PUBLIC] last_sync_state=${JSON.stringify(state)}`);

  for (const tab of tabs) {
    const lastRow = state[`tab_${tab.gid}`] || 0;

    const csvUrl = `https://docs.google.com/spreadsheets/d/${sheetId}/export?format=csv&gid=${tab.gid}`;
    console.log(`[GSHEETS-PUBLIC] Fetching tab="${tab.title}" gid=${tab.gid} lastRow=${lastRow}`);
    
    const resp = await fetch(csvUrl);
    if (!resp.ok) {
      console.warn(`[GSHEETS-PUBLIC] Tab ${tab.title} fetch failed: HTTP ${resp.status}`);
      continue;
    }
    
    const text = await resp.text();
    const rows = parseCsv(text);
    console.log(`[GSHEETS-PUBLIC] Tab "${tab.title}" parsed ${rows.length} total rows`);

    // ─── Header detection: conservative ───────────────────
    const HEADER_WORDS = ['nom', 'name', 'phone', 'telephone', 'tel', 'address', 'adresse',
                          'city', 'ville', 'product', 'produit', 'price', 'prix', 'fullname'];
    let headerMatches = 0;
    for (const cell of (rows[0] || [])) {
      const norm = String(cell).toLowerCase().trim();
      if (HEADER_WORDS.some(w => norm === w)) headerMatches++;
    }
    const row0HasPhoneShape = (rows[0] || []).some(c => /^\+?\d{8,}$/.test(String(c || '').trim()));
    const dataStart = (headerMatches >= 2 && !row0HasPhoneShape) ? 1 : 0;
    
    console.log(`[GSHEETS-PUBLIC] Tab "${tab.title}" header_matches=${headerMatches} has_phone_in_row0=${row0HasPhoneShape} dataStart=${dataStart}`);

    const newRows = rows.slice(Math.max(dataStart, lastRow));
    console.log(`[GSHEETS-PUBLIC] Tab "${tab.title}" newRows to process: ${newRows.length}`);

    let createdCount = 0;
    let skippedCount = 0;
    
    for (let i = 0; i < newRows.length; i++) {
      const row = newRows[i];
      const rowIndex = Math.max(dataStart, lastRow) + i;

      const getCell = (key: string): string => {
        const col = mapping[key];
        if (col === undefined || col === null) return '';
        return (row[col] || '').toString().trim();
      };

      const name = getCell('name');
      const phone = getCell('phone');
      
      console.log(`[GSHEETS-PUBLIC]   Row ${rowIndex + 1}: name="${name}" phone="${phone}"`);

      if (!name && !phone) {
        console.log(`[GSHEETS-PUBLIC]   Row ${rowIndex + 1}: SKIP (empty name and phone)`);
        skippedCount++;
        continue;
      }
      if (!phone) {
        console.log(`[GSHEETS-PUBLIC]   Row ${rowIndex + 1}: SKIP (no phone)`);
        skippedCount++;
        continue;
      }

      const ref = `GSP-${sheetId.slice(0, 6)}-${tab.gid}-R${rowIndex + 1}`;

      // Idempotency check
      const existing = await db.select({ id: orders.id }).from(orders)
        .where(eq(orders.orderNumber, ref))
        .limit(1);
      if (existing.length > 0) {
        console.log(`[GSHEETS-PUBLIC]   Row ${rowIndex + 1}: SKIP (already exists as order ${existing[0].id})`);
        skippedCount++;
        continue;
      }

      const totalPriceCents = Math.round(
        parseFloat(getCell('price').replace(',', '.') || '0') * 100
      ) || 0;
      const product = getCell('product') || tab.title;
      const qty = parseInt(getCell('quantity') || '1', 10) || 1;

      try {
        const order = await storage.createOrder({
          storeId:         conn.storeId,
          magasinId:       conn.magasinId,  // ⚠ CRITICAL: must be set
          orderNumber:     ref,
          customerName:    name,
          customerPhone:   phone,
          customerAddress: getCell('address'),
          customerCity:    getCell('city'),
          rawProductName:  product,
          status:          'nouveau',
          totalPrice:      totalPriceCents,
          productCost:     0,
          shippingCost:    0,
          adSpend:         0,
          source:          'gsheets',
          comment:         getCell('note') || null,
          utmSource:       getCell('utmSource') || null,
          utmCampaign:     getCell('utmCampaign') || null,
        } as any, [{
          rawProductName: product,
          quantity: qty,
          price: totalPriceCents,
        }]);
        
        createdCount++;
        console.log(`[GSHEETS-PUBLIC]   Row ${rowIndex + 1}: ✅ Created order id=${order.id} ref=${ref} magasin=${conn.magasinId || 'NULL'}`);
      } catch (err: any) {
        console.error(`[GSHEETS-PUBLIC]   Row ${rowIndex + 1}: ❌ FAILED to create order: ${err.message}`);
      }
    }

    console.log(`[GSHEETS-PUBLIC] Tab "${tab.title}" summary: created=${createdCount} skipped=${skippedCount}`);
    
    // Only update lastSyncState if we processed all rows (even skipped ones count toward "we saw them")
    state[`tab_${tab.gid}`] = rows.length;
  }

  await db.update(storeIntegrations)
    .set({
      gsheetSyncState: state,
      lastSyncAt: new Date(),
    } as any)
    .where(eq(storeIntegrations.id, conn.id));
  
  console.log(`[GSHEETS-PUBLIC] Updated last_sync_state for connection ${conn.id}`);
}

Fix 6 — Fix the off-by-one bug in state[tab_X]
The previous code stored state[tab_X] = rows.length BEFORE actually attempting order creation. If creation failed (e.g., DB error), the rows are marked as "seen" but no order exists — losing data forever.
The fix above is correct: only mark state[tab_X] = rows.length at the very END after all rows processed. Failed rows will retry next tick because they still have existing.length === 0 (no order with that ref).
This way:

✅ First tick processes all existing rows, creates orders
✅ Second tick sees state[tab_X] = N and only looks at rows N+1 onwards
✅ Failed-to-create rows will retry because the idempotency check passes (order doesn't exist yet)
