2a. Find the OzonExpress tracking sync function (likely in a cron or sync route). Find where it calls the OzonExpress API with the key. Add a check — if the API returns CHECK_API.RESULT === "ERROR", stop the sync for this account and log a clear error:
tsconst data = await response.json();

Find where the sync is triggered. Add:
ts// Skip sync if last attempt failed with invalid key
const lastLog = await db.select().from(integrationLogs)
