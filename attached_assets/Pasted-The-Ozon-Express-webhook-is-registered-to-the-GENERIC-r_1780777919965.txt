The Ozon Express webhook is registered to the GENERIC route
  app.all("/api/webhooks/carrier/:storeId/:carrierName")  in server/routes.ts
but that handler reads rawStatus from (body.status||body.etat||body.statut||body.etat_libelle)
and matches orders via processCarrierWebhook — NEITHER reads Ozon's fields. Ozon sends
{ orderId: "<tracking>", orderStatus: "<CODE>", situation, note }. Result: every Ozon webhook
logs `statut: ""` and "aucune commande trouvée". Fix it by handling Ozon explicitly in this route.

In app.all("/api/webhooks/carrier/:storeId/:carrierName"), RIGHT AFTER the active carrier account
is found (the `account` variable, before the `const rawStatus = ...` line), add an Ozon branch:

    if (carrierName === "ozonexpress") {
      const tracking   = (body.orderId || body.tracking_number || body.code || "").toString().trim();
      const statusCode = (body.orderStatus || body.status || "").toString().trim();
      const note       = (body.note || "").toString();
      console.log(`[OZON-WEBHOOK] store=${storeId} orderId=${tracking} orderStatus=${statusCode}`);

      await storage.createIntegrationLog({ storeId, integrationId: null, provider: "ozonexpress",
        action: "webhook_received", status: "ok",
        message: `Ozon webhook: orderId=${tracking} status=${statusCode}` });

      if (!tracking) return res.json({ success: true, matched: false });
      const order = await storage.getOrderByTrackingNumber(storeId, tracking);
      if (!order) {
        await storage.createIntegrationLog({ storeId, integrationId: null, provider: "ozonexpress",
          action: "webhook_no_match", status: "ok",
          message: `⚠️ Ozon: aucune commande pour orderId=${tracking} — statut: "${statusCode}"` });
        return res.json({ success: true, matched: false });
      }
      const mapped = mapOzonStatus(statusCode);   // already imported from carrier-service
      if (mapped && mapped !== order.status) {
        await storage.updateOrderStatus(order.id, mapped);
        await storage.createOrderFollowUpLog({ orderId: order.id, agentId: null,
          agentName: "Ozon Express Webhook",
          note: `Statut via webhook: ${statusCode} → ${mapped}${note ? " · " + note : ""}` });
        await storage.createIntegrationLog({ storeId, integrationId: null, provider: "ozonexpress",
          action: "status_update", status: "ok",
          message: `✅ Ozon #${order.orderNumber}: ${order.status} → ${mapped} (${statusCode})` });
      }
      return res.json({ success: true, matched: true, newStatus: mapped });
    }

Make sure mapOzonStatus is imported in server/routes.ts from server/services/carrier-service
(the dedicated /api/webhooks/shipping/ozonexpress handler already imports it).
