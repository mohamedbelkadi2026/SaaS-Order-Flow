🚀 FEATURE — User-defined column mapping for Google Sheets public URL
My problem
The public URL connector uses a fixed positional column mapping (A=name, B=phone, C=city, D=address, E=product, F=price, G=quantity). But every user's sheet has columns in a different order — some have phone in column A, some have product in column A, etc.
Currently:

Users with non-standard column order get garbage data imported (wrong field in wrong slot)
Asking them to reorder their existing sheets is friction — they have months of data they don't want to touch

I want users to configure the mapping themselves during connect: a dropdown for each TajerGrow field where they pick which column letter in their sheet contains that data.
How it works
After pasting URL, the user sees a mapping form:

  Champ TajerGrow      →  Colonne de votre sheet
  ────────────────────────────────────────────────
  Nom client *          →  [B ▼]   (ils ont fullname en col B)
  Téléphone *           →  [A ▼]   (phone en col A)
  Adresse               →  [E ▼]
  Ville                 →  [C ▼]
  Produit               →  [D ▼]   (or use tab name if empty)
  Prix                  →  [F ▼]
  Quantité              →  [G ▼]   (default 1 if empty)
  Note                  →  [H ▼]
  UTM Campaign          →  [K ▼]
  UTM Source            →  [L ▼]
  Product ID (Ameex)    →  [M ▼]

* Champs obligatoires

Bonus: dropdown options show the first cell's value as a preview:
  [A ▼]  → "A (ex: 0661639533)"
  [B ▼]  → "B (ex: Haimer Lahcen)"
  
This makes it obvious which column contains what.
The mapping is saved per-integration and used by the cron when fetching CSV rows.

In server/routes.ts, the /api/integrations/google-sheets/connect-url endpoint needs to do TWO phases:

app.post('/api/integrations/google-sheets/preview-url', requireAuth, async (req, res) => {
  const { url } = req.body as { url: string };
  if (!url) return res.status(400).json({ error: 'URL manquante' });

  const sheetId = extractSheetId(url);
  if (!sheetId) {
    return res.status(400).json({
      error: "URL invalide. Collez l'URL complète de votre Google Sheet.",
    });
  }

  let tabs: Array<{ gid: string; title: string }>;
  try {
    tabs = await fetchPublicSheetTabs(sheetId);
  } catch (err: any) {
    return res.status(400).json({
      error: "Impossible d'accéder au sheet. Vérifiez qu'il est partagé avec 'Tout le monde avec le lien' (Lecteur).",
    });
  }

  // Fetch first row of the first tab for preview
  const previewUrl = `https://docs.google.com/spreadsheets/d/${sheetId}/export?format=csv&gid=${tabs[0].gid}`;
  let sampleRow: string[] = [];
  let columnCount = 0;
  try {
    const r = await fetch(previewUrl);
    if (r.ok) {
      const text = await r.text();
      const rows = parseCsv(text);
      // Find the first non-empty row to use as a sample
      sampleRow = rows.find(row => row.some(c => c.trim())) || [];
      columnCount = Math.max(...rows.slice(0, 5).map(r => r.length), sampleRow.length);
    }
  } catch (err) {
    // Non-fatal — proceed without preview
  }

  res.json({
    success: true,
    sheetId,
    tabs,
    sampleRow,
    columnCount,
  });
});

  const { url, magasinId, columnMapping } = req.body as {
    url: string;
    magasinId?: number;
    columnMapping?: Record<string, number>;
  };

  if (!magasinId) {
    return res.status(400).json({
      error: "Veuillez sélectionner un magasin avant de connecter."
    });
  }

  if (!columnMapping || typeof columnMapping !== 'object') {
    return res.status(400).json({
      error: "Veuillez configurer le mapping des colonnes."
    });
  }

  // Mandatory mapping fields
  if (columnMapping.name === undefined || columnMapping.phone === undefined) {
    return res.status(400).json({
      error: "Les colonnes Nom et Téléphone sont obligatoires."
    });
  }

  // ... existing validation (extractSheetId, fetchPublicSheetTabs) ...

  // Persist with the mapping
  await db.update(storeIntegrations)
    .set({
      gsheetUrl: url,
      gsheetId: sheetId,
      gsheetTabs: tabs,
      gsheetColumnMapping: columnMapping,
      gsheetSyncState: {},
      magasinId,
      status: 'active',
    } as any)
    .where(/* ... existing where clause ... */);

  res.json({ success: true, tabsCount: tabs.length, tabs: tabs.map(t => t.title) });
});
Step 3 — Cron: use the user's mapping instead of fixed positional
In server/cron/sync-gsheets-public.ts, replace the hardcoded POSITIONAL constant with reading from the integration:
tsasync function syncOne(conn: any) {
  const sheetId = conn.gsheetId;
  const tabs = conn.gsheetTabs || [];
  const state = conn.gsheetSyncState || {};

  // ⭐ Read user's column mapping (with safe defaults)
  const mapping = conn.gsheetColumnMapping || {
    name: 0, phone: 1, city: 2, address: 3,
    product: 4, price: 5, quantity: 6,
    note: 7, offer: 8,
    utmCampaign: 10, utmSource: 11, productId: 12,
  };

  for (const tab of tabs) {
    const lastRow = state[`tab_${tab.gid}`] || 0;
    const csvUrl = `https://docs.google.com/spreadsheets/d/${sheetId}/export?format=csv&gid=${tab.gid}`;
    const resp = await fetch(csvUrl);
    if (!resp.ok) continue;
    const text = await resp.text();
    const rows = parseCsv(text);

    // Conservative header detection (from previous fix)
    const HEADER_WORDS = ['nom', 'name', 'phone', 'telephone', 'tel', 'address', 'adresse',
                          'city', 'ville', 'product', 'produit', 'price', 'prix', 'fullname',
                          'note', 'comment', 'utm_source', 'utm_campaign', 'sku'];
    let headerMatches = 0;
    for (const cell of (rows[0] || [])) {
      const norm = String(cell).toLowerCase().trim();
      if (HEADER_WORDS.some(w => norm === w || norm.includes(w))) headerMatches++;
    }
    const row0HasPhoneShape = (rows[0] || []).some(c => /^\+?\d{8,}$/.test(String(c).trim()));
    const dataStart = (headerMatches >= 2 && !row0HasPhoneShape) ? 1 : 0;

    const newRows = rows.slice(Math.max(dataStart, lastRow));

    for (let i = 0; i < newRows.length; i++) {
      const row = newRows[i];
      const rowIndex = Math.max(dataStart, lastRow) + i;

      // ⭐ Read using user's mapping
      const getCell = (key: string): string => {
        const col = mapping[key];
        if (col === undefined || col === null) return '';
        return (row[col] || '').toString().trim();
      };

      const name  = getCell('name');
      const phone = getCell('phone');
      if (!name && !phone) continue;
      if (!phone) continue;

      const ref = `GSP-${sheetId.slice(0, 6)}-${tab.gid}-R${rowIndex + 1}`;
      const existing = await db.select({ id: orders.id }).from(orders)
        .where(eq(orders.orderNumber, ref))
        .limit(1);
      if (existing.length > 0) continue;

      const totalPriceCents = Math.round(
        parseFloat(getCell('price').replace(',', '.') || '0') * 100
      ) || 0;
      const product = getCell('product') || tab.title;
      const qty     = parseInt(getCell('quantity') || '1', 10) || 1;

      try {
        await storage.createOrder({
          storeId:         conn.storeId,
          magasinId:       conn.magasinId,
          orderNumber:     ref,
          customerName:    name,
          customerPhone:   phone,
          customerAddress: getCell('address'),
          customerCity:    getCell('city'),
          rawProductName:  product,
          status:          'nouveau',
          totalPrice:      totalPriceCents,
          productCost:     0,
          shippingCost:    0,
          adSpend:         0,
          source:          'gsheets',
          comment:         getCell('note') || null,
          utmSource:       getCell('utmSource') || null,
          utmCampaign:     getCell('utmCampaign') || null,
        } as any, [{
          rawProductName: product,
          quantity: qty,
          price: totalPriceCents,
        }]);
      } catch (err: any) {
        console.error(`[GSHEETS-PUBLIC] Order creation failed ref=${ref}: ${err.message}`);
      }
    }

    state[`tab_${tab.gid}`] = rows.length;
  }

  await db.update(storeIntegrations)
    .set({ gsheetSyncState: state, lastSyncAt: new Date() } as any)
    .where(eq(storeIntegrations.id, conn.id));
}

The ColumnMapField component:
tsxfunction ColumnMapField({ label, value, onChange, required, preview, hint }: {
  label: string;
  value: number | null;
  onChange: (v: number | null) => void;
  required?: boolean;
  preview: { sampleRow: string[]; columnCount: number };
  hint?: string;
}) {
  return (
    <div className="grid grid-cols-12 gap-2 items-center">
      <div className="col-span-5">
        <Label className="text-sm">{label}</Label>
        {hint && <p className="text-[10px] text-muted-foreground">{hint}</p>}
      </div>
      <div className="col-span-7">
        <Select
          value={value === null ? '__none__' : String(value)}
          onValueChange={(v) => onChange(v === '__none__' ? null : parseInt(v, 10))}
        >
          <SelectTrigger>
            <SelectValue placeholder="Pas de colonne" />
          </SelectTrigger>
          <SelectContent>
            {!required && <SelectItem value="__none__">— Pas de colonne —</SelectItem>}
            {Array.from({ length: preview.columnCount }, (_, i) => {
              const letter = columnLetter(i + 1);
              const sample = preview.sampleRow[i]?.slice(0, 30) || '(vide)';
              return (
                <SelectItem key={i} value={String(i)}>
                  {letter} — {sample}
                </SelectItem>
              );
            })}
          </SelectContent>
        </Select>
      </div>
    </div>
  );
}

function columnLetter(col: number): string {
  let s = '';
  while (col > 0) {
    const rem = (col - 1) % 26;
    s = String.fromCharCode(65 + rem) + s;
    col = Math.floor((col - 1) / 26);
  }
  return s;
}
Step 5 — Handlers
tsconst handlePreview = async () => {
  const r = await fetch('/api/integrations/google-sheets/preview-url', {
    method: 'POST',
    credentials: 'include',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ url: sheetUrl }),
  });
  const data = await r.json();
  if (!r.ok) {
    toast({ title: 'Erreur', description: data.error, variant: 'destructive' });
    return;
  }
  setPreview(data);
  setStep('mapping');
};

const handleConnect = async () => {
  // Strip null values from mapping
  const cleanMapping: Record<string, number> = {};
  for (const [key, val] of Object.entries(mapping)) {
    if (val !== null) cleanMapping[key] = val;
  }

  const r = await fetch('/api/integrations/google-sheets/connect-url', {
    method: 'POST',
    credentials: 'include',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      url: sheetUrl,
      magasinId: parseInt(selectedMagasin, 10),
      columnMapping: cleanMapping,
    }),
  });
  const data = await r.json();
  if (!r.ok) {
    toast({ title: 'Erreur', description: data.error, variant: 'destructive' });
    return;
  }
  toast({
    title: '✅ Google Sheets connecté',
    description: `Mapping enregistré. La première synchronisation démarre dans 30 secondes.`,
    duration: 8000,
  });
  setStep('done');
  refetch();
};

Acceptance criteria

✅ Step 1: User pastes URL → click "Suivant" → backend validates + returns sample row.
✅ Step 2: User sees mapping form with dropdowns showing column letter + sample data preview (e.g. "A — 0661639533").
✅ Required fields (Nom, Téléphone) can't be set to "Pas de colonne".
✅ Magasin selector mandatory before "Activer".
✅ Click "Activer" → mapping saved to gsheet_column_mapping column.

✅ User with phone in column A, name in column B → orders import correctly.
