## 0. Plan model (single source of truth)

Treat the **only restricted tier as `starter`**. All other plans (`trial`, `pro`, `elite`, `custom`, or any future tier) get full feature access by default. Define a small helper in `server/storage.ts` (or a new `server/utils/plan.ts`) and reuse it everywhere:

```ts
// Plan capability defaults. Only "starter" is restricted.
export function planDefaults(plan: string) {
  const restricted = plan === 'starter';
  return {
    maxConfirmationAgents: restricted ? 2 : Infinity,  // trial/pro/etc = unlimited
    maxLinkedCarriers:     restricted ? 1 : Infinity,
    automation:            !restricted,                 // starter = false
    mediaBuyers:           !restricted,                 // starter = false
  };
}
```

> Order limits stay as they already are: trial = 60, starter = 1500, pro = unlimited. Do NOT change those numbers.

## 1. Schema — super-admin override columns

In `shared/schema.ts`, in the `subscriptions` table, add two **nullable** override columns:

```ts
automationOverride:  integer("automation_override"),   // null = follow plan, 1 = force ON, 0 = force OFF
mediaBuyersOverride: integer("media_buyers_override"),  // null = follow plan, 1 = force ON, 0 = force OFF
```

Effective access rule (use in both server and the value sent to the client):

```ts
// override wins over the plan default; null means "follow plan"
const hasAutomation  = sub.automationOverride  != null ? sub.automationOverride  === 1 : planDefaults(sub.plan).automation;
const hasMediaBuyers = sub.mediaBuyersOverride != null ? sub.mediaBuyersOverride === 1 : planDefaults(sub.plan).mediaBuyers;
```

Run `npm run db:push`.

## 3. Enforce Starter limit: max 2 confirmation agents

- Load the store subscription, compute `planDefaults(sub.plan).maxConfirmationAgents`.
- If the new user is `role === 'agent'` and its `roleInStore` is `'confirmation'` or `'both'`, count existing agents in this store whose `storeAgentSettings.roleInStore` is `'confirmation'` or `'both'`.
- If count `>=` the max, return HTTP 403 with:
  `{ message: "Le plan Starter est limité à 2 agents de confirmation. Passez au plan Pro pour ajouter plus d'agents." }`
- For non-starter plans the max is `Infinity`, so this never blocks them.

## 4. Enforce Starter limit: max 1 linked delivery company
