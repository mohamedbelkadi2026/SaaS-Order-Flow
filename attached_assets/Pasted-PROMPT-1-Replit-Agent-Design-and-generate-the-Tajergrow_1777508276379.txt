DESIGN — Create the Tajergrow brand logo (master + all favicon sizes)
I want a clean, modern, professional logo for Tajergrow, a Moroccan e-commerce SaaS platform. The logo will appear in:

Browser favicon (16×16, 32×32 — must stay readable at tiny sizes)
iOS / Android home screen icons (180×180, 192×192, 512×512)
Open Graph preview (1200×630 for WhatsApp / Facebook share)
Login screen, sidebar header, emails

Brand identity

Name: Tajergrow (تاجر = "trader" in Arabic + "grow" = growth)
Sector: SaaS e-commerce / order management / Cash-on-Delivery in Morocco
Audience: Moroccan dropshippers, Shopify/YouCan store owners, agencies
Personality: Professional, modern, trustworthy, locally-rooted but tech-forward

Brand palette (USE EXACTLY THESE COLORS)

Primary navy:  #1E1B4B  (RGB 30, 27, 75) — backgrounds, "T" letter mass
Accent gold:   #C5A059  (RGB 197, 160, 89) — growth arrow, highlights
White:         #FFFFFF  (text on navy if needed)

Do NOT introduce any other color. Two-color (navy + gold) is mandatory for a recognizable brand.
Concept
A geometric symbol that fuses two ideas:

A bold letter "T" — solid, blocky, sans-serif. The vertical stem of the T should be thick and grounded.
An upward growth arrow — built INTO the T. The horizontal bar of the T tapers up and to the right into a gold arrowhead, suggesting upward business growth. Think: the top-right corner of the T morphs into an arrow.

Imagine an arrow rising diagonally from the T's vertical stem, breaking out of the top-right of the T's crossbar. The arrow is the gold accent. The T is the navy mass.
The whole symbol fits inside a rounded square (corner radius ~14% of the icon size) with the navy color as background. The T and arrow are gold/white inside.

Approach
You can either:

(a) Generate the master SVG directly using SVG primitives (<rect>, <path>, <polygon>) — recommended, gives crisp results at every size.
(b) Use a tool like sharp or canvas in a script to rasterize an SVG into all the PNG sizes.

If you go with (a), here's a starting SVG skeleton you can iterate on (modify the <path> for the arrow until it looks right):

Iterate this SVG until the design is balanced and crisp at 16×16. Then generate all 11 deliverables.
Generation script
Create scripts/generate-favicons.mjs:
jsimport sharp from 'sharp';
import { mkdirSync, readFileSync } from 'fs';
import { join } from 'path';

mkdirSync('client/public', { recursive: true });

// Read the master SVG (your hand-tuned source)
const svg = readFileSync('client/public/logo-master.svg');

const sizes = [
  { name: 'favicon-16x16.png',      w: 16,   h: 16,  bg: 'transparent' },
  { name: 'favicon-32x32.png',      w: 32,   h: 32,  bg: 'transparent' },
  { name: 'favicon-48x48.png',      w: 48,   h: 48,  bg: 'transparent' },
  { name: 'favicon.png',            w: 192,  h: 192, bg: 'transparent' },
  { name: 'apple-touch-icon.png',   w: 180,  h: 180, bg: '#1E1B4B' },
  { name: 'android-chrome-192.png', w: 192,  h: 192, bg: 'transparent' },
  { name: 'android-chrome-512.png', w: 512,  h: 512, bg: 'transparent' },
  { name: 'logo-master.png',        w: 1024, h: 1024,bg: 'transparent' },
];

for (const { name, w, h, bg } of sizes) {
  await sharp(svg)
    .resize(w, h, {
      fit: 'contain',
      background: bg === 'transparent'
        ? { r: 0, g: 0, b: 0, alpha: 0 }
        : { r: 30, g: 27, b: 75, alpha: 1 },
    })
    .png()
    .toFile(join('client/public', name));
  console.log(`✓ ${name}`);
}

// og-image.png: 1200×630 with logo centered-left + wordmark to its right
await sharp({
  create: {
    width: 1200, height: 630, channels: 4,
    background: { r: 30, g: 27, b: 75, alpha: 1 },
  }
})
  .composite([
    { input: await sharp(svg).resize(400, 400).png().toBuffer(), top: 115, left: 100 },
    // wordmark: render text to SVG, then overlay
    {
      input: Buffer.from(`
        <svg xmlns="http://www.w3.org/2000/svg" width="650" height="200">
          <text x="0" y="100" font-family="Inter, sans-serif" font-size="96" font-weight="800" fill="white">Tajergrow</text>
          <text x="0" y="160" font-family="Inter, sans-serif" font-size="32" font-weight="500" fill="#C5A059">Gestion e-commerce au Maroc</text>
        </svg>
      `),
      top: 215, left: 540,
    },
  ])
  .png()
  .toFile('client/public/og-image.png');
console.log(`✓ og-image.png`);
