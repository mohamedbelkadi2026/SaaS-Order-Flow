Item 2: قفازات التنظيف (upsell) — 49 DH

In Tajergrow platform:

Only 1 product shown
Total: 349 DH (missing the 49 DH upsell)

The total to match the carrier total exactly

Step 7 — Match total to Shopify exactly
Check parsed.totalPrice (line 362 of routes.ts):
tsconst totalPrice = Math.round(parseFloat(payload.total_price || '0') * 100);
Shopify's total_price includes shipping + tax. If you want the platform to show the same 407.80 that Shopify shows, this is already correct — it captures the full total.
If the platform shows 349.00 while Shopify shows 407.80, that means either:

The webhook payload's total_price is wrong (rare — Shopify is reliable here)
We're reading subtotal_price somewhere instead of total_price (search for both)

Add a log:
tsconsole.log(`[SHOPIFY-WEBHOOK] storing totalPrice=${parsed.totalPrice} (= ${parsed.totalPrice/100} DH)`);
If it logs 40780, the storage is right and the UI is wrong. If it logs 34900, the parser is wrong.

async function enrichOrderFromShopify(orderId: string, shop: string, accessToken: string) {
