STEP 0 — DIAGNOSE FIRST (read-only, paste the result back):
Run a query on the affected orders to see where they come from:
  SELECT o.source, COUNT(*) AS n,
         SUM(CASE WHEN o.raw_product_name IS NOT NULL AND o.raw_product_name <> '' THEN 1 ELSE 0 END) AS has_order_name,
         SUM(CASE WHEN oi.id IS NOT NULL THEN 1 ELSE 0 END) AS has_items
  FROM orders o
  LEFT JOIN order_items oi ON oi.order_id = o.id
  WHERE o.track_number IS NOT NULL AND o.track_number <> ''
  GROUP BY o.source
  ORDER BY n DESC;
